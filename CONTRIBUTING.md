# Contributing to TreeMc

感谢你愿意参与 TreeMc。

## 提交前检查

- 使用清晰的提交信息，建议采用 `type: summary` 风格
- 保持改动聚焦，避免一次提交混入无关内容
- 文档改动请同步更新 README 或相关说明

## 贡献流程

1. Fork 本仓库并创建功能分支
2. 完成开发与本地验证
3. 提交 Pull Request，描述改动动机与影响范围
4. 等待维护者 Review 并按建议调整

## Pull Request 建议内容

- 改动背景与目标
- 核心实现说明
- 兼容性与风险点
- 测试或验证结果

## 行为准则

请保持沟通友好、尊重他人并聚焦问题本身。
